# drupalx-marketing

Single-page marketing site for [DrupalX](https://github.com/nextagencyio/drupalx-project) —
Astro 5 + Tailwind v4 (CSS-first tokens matching the DrupalX starter brand),
astro-icon (Lucide), self-hosted Inter + Fraunces variable fonts.

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # static dist/
```

Everything lives in `src/pages/index.astro` (the one page) and
`src/styles/global.css` (the brand tokens + design primitives). Screenshots
in `public/img/` are real captures of the DrupalX starter + Canvas editor.
